# 简历模块 — AI模拟面试系统

## 模块定位

简历模块是 AI 模拟面试系统的**个性化输入引擎**。它将用户的技能、经验、求职意向结构化为统一格式（`parsed_data`），驱动面试出题、诊断分析、能力测绘三大核心场景，实现从"通用面试"到"针对你简历的定制面试"的体验升级。

## 核心信条

> **一份简历，两种入口，同一格式，全域复用。**

## 模块边界

```
用户
  ├── 上传简历(PDF/Word/图片) ──→ AI解析 ──→ parsed_data(JSON)
  └── 在线填写(结构化表单) ─────────┘
                                           ↓
  ┌─────────────────────────────────────────────────────┐
  │                    消费方                             │
  │  InterviewContextService  │  PromptBuilder           │
  │  AIDiagnosisService       │  SkillVerification      │
  │  ResumeComparision        │  ...                    │
  └─────────────────────────────────────────────────────┘
```
